# DepotMap - Digitális Raktártérkép

## Leírás
Webalapú raktárkezelő kis- és középvállalkozásoknak. Kétszintű rácshálós raktártervező polc/rekesz nyilvántartással, beszerzés/rendelés kezeléssel.

## Cél
- Excel/papír nyilvántartás hibáinak kiküszöbölése
- Termékek pontos helyének megadása
- Raktár átrendezés egyszerűsítése
- WMS alternatíva

## Fő funkciók
- Raktár alaprajz szerkesztése rácshálóban
- Polcok és rekeszek nyilvántartása
- Termék készletkezelés rekesz szinten
- Beszerzés és rendelés kezelés
- Role alapú jogosultságok

## Tech stack
- Frontend: Angular + Bootstrap
- Backend: .NET
- Adatbázis: MSSQL
